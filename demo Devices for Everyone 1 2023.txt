Nombre de la aplicación: Devices for Everyone.
•
Dirigida a estudiantes y compañeros de nuestra universidad.
Diapositiva 2:
Problema a resolver:
david
•
Según estimaciones del Banco Mundial de Discapacidad, aproximadamente el 15% de la población mundial, es decir, mil millones de personas, sufren de algún tipo de discapacidad.
•
Nuestra aplicación está diseñada para ayudar a las personas con discapacidad a encontrar y comprar dispositivos adaptados a sus necesidades especiales.
Diapositiva 3:
Objetivos de la aplicación:
brisa
1. Facilitar la búsqueda rápida y eficiente de productos adaptados.
2. Permitir a los usuarios encontrar productos acordes a su tipo de discapacidad.
3. Brindar a las empresas una plataforma para publicar sus productos adaptados en una sola tienda.

Diapositiva 6:
brisa
Investigación tecnológica:
•
Durante el proceso, tuvimos que tomar decisiones tecnológicas clave.
•
Consideramos diferentes opciones como Node.js y Spring para el backend, y Spring JDBC y Java JDBC para la conexión con la base de datos.
•
Decidimos utilizar Spring y Java JDBC para el backend y MySQL para la base de datos.
